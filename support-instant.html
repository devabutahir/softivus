Support Policy

Whats Included
Assistance With Theme Installation

If after following all instructions in theme documentation you still have troubles installing your theme, well gladly help you and point you in the right direction.

Modifications Covered by Theme Functionality

Minor tweaks orchanges thatcan be done without modifying theme source files or implementing new features.

Theme Compatibility With Included Plugins

We guarantee that your theme works as intended with any plugin bundled with or recommended by the theme. In case any issue occurs, it will be fixed in a timely manner.

Assistance With Reported Theme Bugs

Contact our support team if youve found an obvious bug that prevents you from using the theme as intended.

Providing Information about Recommended Server Configuration

The list of theme requirements is available in the theme documentation. Our Tech Support Team can always point you to the server configuration changes that will solve the issue you faced.

General Guidelines on Website Speed Optimization

Website loading time depends on many factors like server response time, image optimization, caching etc. Our Tech Support Team can check your website and point you to the steps on improving your website performance.

Whats Not Included
Complete Theme Installation From Start to Finish

If youre new to WordPress, or simply dont have enough time, you can always order our theme installation service and well install the theme for you.

Customization Not Covered by Theme Functionality

This includes extending or modifying your theme beyond its advertised features. For any development work, you can always refer to our custom studio.

Support for 3rd Party Plugins

We do not guarantee full compatibility of your theme with 3rd party plugins. Making a theme compliant with the plugin may require additional customization which is not included in support service.

Help With Additional Scripts or Custom Code Added to the Theme

Custom code and scripts you add on the website may conflict with the theme.Resulting issues are not covered by our Support Policy.For any development work, you can always refer to our custom studio.

Server Environment Setup

Issues caused by your web hosting, server environment or software installed. Please contact your web hosting provider or consult the documentation for the software.
